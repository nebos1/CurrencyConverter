Simple Window app made on Java for currency converter.

NOTE: Converter up to date 06.10.2025 (dd/mm/yy).


Supported currencies: 

Bulgarian lev - BGN
Euro - EUR
American dollar - USD
British lire - GBP
Turkish lire - TRY
Russian ruble - RUB
Chinese yuan - CNY
Serbian dinar - RSD
Makedonian denar - MKD
Japanese yen - JPY
